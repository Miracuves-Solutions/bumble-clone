## How It Works

1. User creates a profile with photos, bio, interests, and verification
2. Discovery algorithm presents potential matches based on preferences and location
3. User swipes right to like or left to pass on each profile
4. When both users swipe right, it is a match
5. In women-first mode, only women can send the first message
6. Matched users chat, video call, and potentially meet
